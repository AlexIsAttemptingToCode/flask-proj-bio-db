Help & Documentation

How to Search

You can search the database using two methods:

  • UniProt ID: Enter a specific ID (e.g., P0DTC2). If the protein is not in our database, the system will attempt to download it automatically.
  • Protein Name: Enter a keyword (e.g., Spike or Insulin).
API Usage

Developers can access raw JSON data via our REST API.

Endpoint: /api/protein/<ID>

Example: /api/protein/P0DTC2
